# 🧠 Quiz Game

A simple command-line quiz game built using Python and Object-Oriented Programming (OOP).

## Features

* Multiple-choice quiz questions
* Score tracking
* Object-Oriented Programming structure
* Easy to add new questions

## Project Structure

```text
quiz-game/
│
├── main.py
├── question_model.py
├── quiz_brain.py
└── data.py
```

## Concepts Used

* Classes and Objects
* Constructors (`__init__`)
* Loops
* Lists
* Conditional Statements
* Modular Programming

## How to Run

```bash
python main.py
```

## Example

```text
Q. The sky is blue? (True/False): True
You got it right!
Current Score: 1/1
```
